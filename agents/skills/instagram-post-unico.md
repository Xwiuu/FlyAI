# Skill — Instagram Post Único (Reels / Estático)

## Objetivo
Legenda de alta conversão para Reels ou imagem estática. Uma ideia, um argumento, um CTA.

## Estrutura
- Hook (linha 1): aparece no preview antes do "mais" — para o scroll ou não.
- Body: 1–3 parágrafos de 2–4 linhas. Cada parágrafo entrega prova ou avanço.
- Hashtags: 0–5, técnicas, ao final.
- CTA: último parágrafo, específico, sem urgência fabricada.

## Comprimento
- 200–500 caracteres no body (sem contar hook e CTA).
- Parágrafos separados por linha em branco (\n\n).
- Sem listas ou bullets — prosa direta.

## Regras
- A primeira palavra do hook deve criar tensão ou citar um número.
- Nunca começar com "Você sabia" ou "Hoje vamos falar sobre".
- CTA deve oferecer especificidade ("respondo no DM com o template") — nunca genérico ("curta e compartilhe").

# Skill — Instagram Stories

## Objetivo
Sequência de 3–5 story frames coesos que criam engajamento via contexto, bastidores ou pergunta rápida. Alta frequência, baixa barreira de entrada.

## Estrutura por frame
- Frame 1: contexto ou gancho de bastidores (máx 70 chars).
- Frames 2–N-1: desenvolvimento em unidades mínimas — 1 ideia por frame.
- Frame final: enquete, caixa de perguntas ou CTA para DM.

## Regras
- Cada frame: máximo 80 caracteres (legibilidade no mobile sem zoom).
- Use linguagem direta, sem subordinadas longas.
- Pelo menos 1 frame deve ser interativo (enquete / "responda aqui" / pergunta fechada).
- Nunca inclua links (stories orgânicos sem swipe-up padrão).
- Tom: bastidor honesto ou tensão revelada — nunca tutorial genérico.
